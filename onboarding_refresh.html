Configuration à Finaliser

La configuration de votre compte Stripe n'est pas encore terminée. Pour recevoir des paiements via Babacoiffure, vous devez compléter toutes les étapes du processus d'inscription.

Pourquoi est-ce important ?

ℹ️ Recevoir des paiements

Pour pouvoir recevoir des paiements de vos clients via notre plateforme, vous devez disposer d'un compte Stripe correctement configuré.

ℹ️ Les informations requises

Pour finaliser la configuration, vous devrez fournir :

  • • Vos informations personnelles
  • • Les détails de votre entreprise (si applicable)
  • • Vos coordonnées bancaires pour recevoir vos fonds
  • • Une pièce d'identité valide pour vérification

ℹ️ Sécurité et confidentialité

Vos informations sont transmises directement à Stripe de manière sécurisée. Babacoiffure n'a pas accès à vos informations bancaires ou à vos documents d'identité.

ℹ️ Besoin d'aide ?

Si vous rencontrez des difficultés pendant le processus d'inscription, n'hésitez pas à contacter notre support à admin@babacoiffure.com.
